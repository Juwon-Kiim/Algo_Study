#  BOJ_G5_2225_합분해 [2023.02.07]
https://www.acmicpc.net/problem/2225

<접근법>

```
1. N을 K개의 숫자로 표현하기 위해, 0~N까지의 숫자를 0~K-1개의 수로 나타내는 방법을 dp로 구한다.
2. 예를 들어, 20이라는 숫자를 2개의 숫자로 표현하려면, 첫번째 숫자는 20이하의 수인 0~20으로 표현할 수 있고 그 뒷 숫자는 20에서 첫번째 숫자를 뺀 수를 한 개의 숫자로 표현한 것이다.
3. 모든 수는 한 개의 숫자로 나타낼 수 있는 방법이 1가지이기 때문에(자기 자신) 이를 통해 2개의 숫자로 표현하는 방법을 구할 수 있다.
4. 이를 식으로 표현한다면,
    dp[i][j] = dp[i-1][j] + dp[i-1][j-1] + ... + dp[i-1][0] = dp[i-1][j] + dp[i][j-1]
```



```python
'''
* 메모리 : 33300 KB
* 시간 : 52 ms
'''
N, K = map(int, input().split())

dp = [[1] * (N+1) for _ in range(K+1)]

for i in range(2, K+1):
    for j in range(1, N+1):
        dp[i][j] = dp[i][j-1] + dp[i-1][j]
print(dp[K][N] % 1000000000)
```
