# BOJ_G5_15486_퇴사 2 [2023.01.31]

<접근법>
```
원래 앞에서부터 접근해서 풀려고 했으나, 예외 사항이 너무 많아서 풀 수가 없었습니다
그래서 뒤에서부터 접근해서, 해당 상담을 하는 경우와 하지 않는 경우로 나눠 둘 중 더 큰 값을 계속 받아오는 방법을 취했습니다.
```



---

# Python




```python
'''
* 메모리 : 291276 KB
* 시간 : 1080 ms
'''

import sys
input = sys.stdin.readline

N = int(input())
dp = [0] * (N+1)
lst = [list(map(int, input().split())) for _ in range(N)]

# 해당 상담을 했을 경우, 하지 않았을 경우 중 최댓값을 찾아 갱신해나감
for i in range(N-1, -1, -1):
    T, P = lst[i]

    # 해당 상담을 퇴사 전까지 할 수 있으면서 하는 경우가 이득인 경우
    # 해당 상담을 퇴사 전까지 못 하거나 안 하는 경우가 이득은 경우
    dp[i] = P + dp[T+i] if T + i <= N and P + dp[T+i] > dp[i+1] else dp[i+1]

print(dp[0])
```

# Java




```java
/**
 * 메모리  : 442228 KB
 * 시간   : 1208 ms
 */

import java.io.*;
import java.math.*;

public class Main {

    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int N = Integer.parseInt(br.readLine());
        int[][] lst = new int[N][2];
        String[] temp;
        for (int i = 0; i < N; i++) {
            temp = br.readLine().split(" ");
            lst[i][0] = Integer.parseInt(temp[0]);
            lst[i][1] = Integer.parseInt(temp[1]);
        }

        // 최대로 나올 수 있는 값이 1,500,000,000 이므로 int 범위를 벗어날 수 있다.
        // 따라서 BigInteger를 쓴다
        BigInteger[] dp = new BigInteger[N+1];
        dp[N] = new BigInteger("0");
        
        int T;
        BigInteger P;

        // 해당 상담을 했을 경우, 하지 않았을 경우 중 최댓값을 찾아 갱신해나감
        for (int i = N-1; i > -1; i--) {
            T = lst[i][0];
            P = BigInteger.valueOf(lst[i][1]);

            // 해당 상담을 퇴사 전까지 할 수 있으면서 하는 경우가 이득인 경우
            if (T+i <= N && P.add(dp[T+i]).compareTo(dp[i+1]) > 0) {
                dp[i] = P.add(dp[T+i]);
            
            // 해당 상담을 퇴사 전까지 못 하거나 안 하는 경우가 이득인 경우
            } else {
                dp[i] = dp[i+1];
            }
        }
        System.out.println(dp[0]);
    }
}
```
