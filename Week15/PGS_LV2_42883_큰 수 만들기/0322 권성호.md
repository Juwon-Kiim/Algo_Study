#   [2023.03.22] PGS_LV2_42883_큰 수 만들기
https://school.programmers.co.kr/learn/courses/30/lessons/42883

<접근법>

```
- stack 자료구조를 사용해 조건을 충족하면 문자(char)을 담는다.
- 마지막으로 스택에 추가된 숫자가 가장 큰 값이 됩니다. 
- 따라서, 스택에 추가되는 숫자가 현재 가장 큰 값보다 작은 경우, 스택에서 해당 숫자를 제거하면 됩니다.

1. 빈 스택을 생성합니다.
2. 스택이 비어있지 않고, 스택의 마지막 값이 현재 문자보다 작을 때까지 스택에서 값을 제거하며, 제거한 값의 개수를 k에서 차감합니다.
3. 현재 문자를 스택에 추가합니다.
4. 만약 k가 0보다 크다면, 스택의 뒤에서 k개의 값을 제거합니다. 이는 스택에서 가장 큰 숫자를 만들기 위해 뒤에서부터 k개의 작은 숫자를 제거하는 것을 의미합니다.
5. 스택에 남은 숫자들을 문자열로 변환하여 반환합니다.

```





```js
/**
 * 메모리  : 33.4 MB
 * 시간   :  0.13 ms
 */

function solution(number, k) {
// 1. 빈 스택을 생성합니다.
    let stack = [];
    for(let i = 0; i < number.length; i++) {
    // 2. 스택이 비어있지 않고, 스택의 마지막 값이 현재 문자보다 작을 때까지 스택에서 값을 제거하며, 제거한 값의 개수를 k에서 차감합니다.
    // 만약 k가 0보다 크다면, 스택의 뒤에서 k개의 값을 제거합니다. 이는 스택에서 가장 큰 숫자를 만들기 위해 뒤에서부터 k개의 작은 숫자를 제거하는 것을 의미합니다.
        while(k > 0 && stack[stack.length-1] < number[i]) {
            stack.pop();
            k--;
        }
        // 현재 문자를 스택에 추가합니다.
        stack.push(number[i]);
    }
    stack.splice(stack.length-k , k);   
    return stack.join('');
}
```
