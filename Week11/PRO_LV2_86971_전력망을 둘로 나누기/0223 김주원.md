# PRO_LV2_86971_전력망을 둘로 나누기 [2023.02.23]

https://school.programmers.co.kr/learn/courses/30/lessons/86971

<접근법>

```
1. 송전탑의 연결 상태를 담고 있는 wires 배열을 가지고 arr[i][j] 배열에 넣어준다
2. arr[i][j]는 i번 송전탑과 j번 송전탑의 연결 상태를 나타낸다
3. 끊을 전선을 선택하여 끊고 dfs를 통해 1번 송전탑과 연결된 송전탑의 개수를 센다
4. dfs를 끝내고 나오면 두 전력망으로 나뉜 각각의 송전탑 개수를 구할 수 있다
5. 한쪽 송전탑 개수를 구했으니 총 송전탑 개수에서 한쪽 송전탑 개수를 뺀 게 다른쪽 송전탑 개수가 된다. 결국 한쪽 송전탑 개수만 알면 (총 송전탑 개수 - 한쪽 송전탑 개수*2)를 통해 답을 구할 수 있다
```



```java
// 시간 : 5.355 ms
// 메모리 : 77.3 MB
class Solution {
    static int cnt;
    static int[][] arr;
    public int solution(int n, int[][] wires) {
        // 연결 상태를 1로 표시한 배열
        arr = new int[n+1][n+1];
        
        int min = Integer.MAX_VALUE;
        
        // 연결
        for(int i=0;i<wires.length;i++){
            arr[wires[i][0]][wires[i][1]] = 1;
            arr[wires[i][1]][wires[i][0]] = 1;
        }
        
        // 끊을 전선 고르기
        for(int i=1;i<n;i++){
            for(int j=i+1;j<=n;j++){
                // 연결되어 있는 줄 발견
                if(arr[i][j] != 0){
                    // 끊기
                    arr[i][j] = 0;
                    arr[j][i] = 0;
                    
                    // 연결된 송전탑 개수
                    cnt = 1;
                    // 1과 연결된 송전탑 dfs
                    // 파라미터 : 송전탑 1번에서 시작, 송전탑의 총 개수
                    dfs(1, n, new boolean[n+1]);
                    min = Math.min(min, Math.abs(n-cnt*2));
                    // 다시 연결
                    arr[i][j] = 1;
                    arr[j][i] = 1;
                }
            }
        }
        
        return min;
    }
    
    static void dfs(int start, int n, boolean[] visited){
        // 현재 송전탑 방문처리
        visited[start] = true;
        // 2번 송전탑부터 시작
        for(int i=2;i<=n;i++){
            // 방문하지 않았고 i번째 송전탑이 start번째 송전탑과 연결되어 있으면 dfs
            if(!visited[i] && arr[start][i]!=0){
                cnt++;
                dfs(i, n, visited);
            }
        }
    }
}
```
