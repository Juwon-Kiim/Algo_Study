# BOJ_S1_10844_쉬운 계단 수 [2023.02.01]
https://www.acmicpc.net/problem/10844

<접근법>
```
0. 제일 첫번째 자릿수가 0부터 9까지 시작하는 숫자를 개수를 센다.(마지막에 0으로 시작하는 수 제외)
1. 한 자릿수인 경우, 0~9까지 모두 1개이다.
2. 두 자릿수 이상인 경우, 두번째 자릿수는 첫번째 자릿수와 -1, +1 차이가 나야하는데, 이 두 개의 수로 시작하는 수 중에서 길이가 현재보다 하나 작은 수의 개수를 더하여 저장한다. 가령, 2**의 개수를 헤아리고 싶다면, 21*과 23*의 개수를 헤아려야하고 이 개수는 결국 1*과 3*의 개수와 동일하다.
3. 0과 9의 경우는 -1한 수와, +1 한 수가 존재하지 않기 때문에 이에 유의하여 코드를 짠다.
```


```python
'''
* 메모리 :  31256KB
* 시간 :  44ms
'''
N = int(input())

dp = [[1] * 10]

for _ in range(N-1):
    new_dp = [0] * 10
    for j in range(10):
        left = dp[-1][j-1] if j > 0 else 0
        right = dp[-1][j+1] if j < 9 else 0
        new_dp[j] = left + right
    dp.append(new_dp)
print(sum(dp[-1][1:]) % 1000000000)
```
